## Pruebas

- Ejecuta pruebas unitarias usando junit:
  ```
  mvn test
  ```
- Ejecuta pruebas de integración usando Spring Boot:
  ```
  mvn verify
  ```
- Para pruebas de automatización usando Selenium:
  ```
  npm test
  ```
